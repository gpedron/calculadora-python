# calculadora-python
Aula Ebac

Descrição

Este projeto consiste em uma calculadora desenvolvida em Python que realiza as quatro operações matemáticas básicas:

Soma
Subtração
Multiplicação
Divisão

O usuário escolhe a operação desejada através de um menu interativo no terminal.


## Executando pelo script

Conceda permissão de execução:

chmod +x calculadora.sh

Execute:

./calculadora.sh
